# Analiza: otroške knjige
projekna naloga pri predmetu Programiranje 1

## Zajeti podatki

Podatke bomo zajeli iz treh razliènih list na spletni strani www.goodreads.com in sicer iz vsake prvih 1200 knjig.
Zajeti podatki so:

* naslov knjige
* avtor
* ocena
* število glasov
* skupna ocena (odvisna od ocene, št. glasov, ...)

## Predvidena analiza

Ogledali si bomo:

* kateri avtor je najbolje ocenjen na posamezni listi in na skupni listi (sestavljeni iz vseh treh list)
* kateri naslov je najbolje ocenjen
* kateri avtor ima na listah najveè knjig
* kako pada priljubljenost avtorjev glede na oceno
* kako je skupna ocena odvisna od števila glasov, ocene
* ...
